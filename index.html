A2G 배송지연 보상 관리 어드민 (PRD 시안)

A2G(셀렉티) 상품의 배송지연 보상 어드민 PRD 시안입니다.
한반도 어드민(G마켓 내부 어드민) 디자인시스템 기반으로 작성되었습니다.

1. 배송지연 보상 내역 관리
개별 보상 건의 조회, 수동 보상 확정, 수동 보상 제외(송장기준/주문기준) 처리 화면. 도착보장일 초과 건의 쿠폰 발행 대상을 운영자가 검토하고 처리합니다.
2. 배송지연 보상 집계 내역
일자별 보상 집계 현황 조회. 도착보장 합계, 보상 확정 건수, 미보상 건수, 쿠폰 발행 결과(송장/주문), 도착보장율 등을 한눈에 볼 수 있습니다.
참고
• 본 화면은 PRD 산출물로, 실제 운영 데이터가 아닌 샘플 데이터입니다.
• 쿠폰 발행 금액: 송장당 1,500원 (스마일캐시 → 쿠폰으로 정책 변경)
• 보상 비용 주체: G마켓 전액 (택배사 귀책 미배달코드도 G마켓 부담)
• 본 어드민에서는 쿠폰 발행 건수만 집계되며, 쿠폰 사용 여부는 관리하지 않습니다.
